# Third-Party Notices

This repository contains original Python source code for a SolidWorks batch
export utility.

## SolidWorks

The project automates an installed local SolidWorks application through the
SolidWorks COM API. It does not bundle SolidWorks, SolidWorks SDK files,
Dassault Systemes software, sample CAD files, or vendor documentation.

Users must have their own valid SolidWorks installation and license to use the
conversion features.
